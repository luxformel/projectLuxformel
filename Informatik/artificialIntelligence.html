Definition

Conventional algorithm

A conventional algorithm can be programmed by the following steps:

  1. We take in the known data

  2. We manually define rules for the algorithm to operate with in

  3. We obtain a result

Smart algorithm

A smart algorithm can be defined in these steps:

  1. We take in the known data

  2. We automatically define our rules through unknown data

  3. We obtain a result
